# Sistema eléctrico

## Estado general observado
La instalación eléctrica actual es artesanal y se encuentra funcional pero expuesta. Hay potencial claro de mejora en seguridad, orden, mantenimiento y diagnóstico.

## Componentes identificados
- batería (ubicación exacta a confirmar)
- alternador
- motor de arranque / burro
- distribuidor
- bobina
- tablero artesanal
- llaves/teclas de comando
- cableado principal y secundario
- masas a relevar
- cortacorriente pendiente de instalación

## Objetivo del rediseño/documentación
- separar circuitos por función
- identificar positivos permanentes y bajo llave
- definir puntos de masa confiables
- proteger cableado
- agregar fusibles o fusiblera si no existieran
- incorporar cortacorriente principal

## Circuitos a documentar
1. arranque
2. carga
3. encendido
4. instrumental
5. iluminación
6. accesorios

## Esquema lógico base propuesto
- batería
  - positivo principal a cortacorriente
  - desde cortacorriente a arranque/distribución
- arranque
  - positivo grueso al solenoide
  - señal desde llave de arranque
- alternador
  - salida de carga a batería o barra principal
- encendido
  - alimentación bajo llave a bobina
  - distribuidor hacia bujías
- tablero
  - alimentación bajo llave
  - reloj(es)
  - indicadores
  - interruptores

## Recomendaciones
- etiquetar cada cable
- usar colores consistentes por circuito
- proteger con corrugado o malla
- evitar recorrido cercano a escape o zonas de roce
- asegurar masas limpias y firmes
- documentar con fotos antes y después de intervenir

## Pendientes de relevamiento
- ubicación exacta de batería
- presencia o ausencia de fusiblera
- qué tecla controla cada circuito
- recorrido de luces delanteras y traseras
- ubicación propuesta para el cortacorriente
