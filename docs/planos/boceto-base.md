## Recomendación de método
1. sacar foto lateral lo más perpendicular posible
2. medir una referencia real fácil, por ejemplo diámetro de rueda
3. usar esa referencia para escalar bocetos
4. completar estas cotas mínimas
5. pasar a draw.io, LibreCAD o FreeCAD
