# Proyecto-Arenero

Documentación técnica, evolución, ideas y versionado del arenero.

## Objetivo del repositorio

Este repo nace para:
- centralizar información del arenero
- documentar componentes mecánicos y eléctricos
- registrar avances, cambios y decisiones
- guardar fotos de referencia
- preparar bocetos y futuros planos con medidas reales
- dejar historial versionado del proyecto

## Estado actual resumido

### Avances ya realizados
- arreglo de frenos
- cambio/revisión de cáliper
- cambio de cañería de freno
- cambio de bomba de freno
- compra de manguera para conexión al tanque de nafta
- materiales listos para instalar cortacorriente
- materiales de cableado, fichas y abrazaderas disponibles

### Base mecánica identificada
- motor Ford 6 cilindros en línea, probablemente familia Falcon/F100 221 o 3.6
- caja ZF
- tracción trasera
- diferencial trasero rígido
- suspensión delantera independiente adaptada/artesanal
- tablero e instalación eléctrica artesanal
- carburador simple boca con adaptación sobre múltiple

## Uso previsto del arenero
- arena
- playa
- caminos difíciles
- uso recreativo off-road
- posible interés futuro en tiradas ocasionales de 1/8 de milla

## Contenido del repositorio

- `docs/fotos/` fotos base del proyecto
- `docs/tecnico/` fichas técnicas y relevamientos
- `docs/sistemas/` documentación por sistema
- `docs/planos/` bocetos, plantillas y futuros planos
